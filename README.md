# 🤖 MateAI – Your Learning Chatbot

MateAI is a simple Python chatbot that can remember facts, answer questions, greet users, and even perform basic calculations.  
It stores learned information in a JSON file (`memory.json`), so it keeps its knowledge between runs.

---

## ✨ Features
- 🧠 **Memory** – Remembers facts you teach it (saved in `memory.json`).  
- ❓ **Q&A** – Can answer questions like "Who is Elon Musk?" once taught.  
- 💬 **Greetings** – Friendly introduction and natural conversation.  
- 🧮 **Calculator** – Handles basic arithmetic (add, subtract, multiply, divide).
